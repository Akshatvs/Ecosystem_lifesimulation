# 2D Ecosystem Simulation in C / SDL2

A real-time 2D ecosystem simulation built in C using SDL2.
The simulation features multiple species with different attributes, food sources, predator-prey interactions, reproduction, inherited traits, mutations, adjustable speed, pause mode, and interactive entity inspection.

## Features

- Real-time 2D simulation
- Three species with different behaviors
- Food, hunger, health and aging systems
- Predator-prey interactions
- Reproduction with inherited traits and random mutation
- Adjustable simulation speed
- Pause and step-by-step mode
- Interactive stats panel for selected individuals
- Custom pixel-art rendering
- SDL2 image, text and audio support

## Technologies

- C
- SDL2
- SDL_ttf
- SDL_image
- SDL_mixer

## Controls

- `ENTER`: start simulation / step-by-step mode
- `SPACE`: pause or resume
- `Z`: increase simulation speed
- `A`: decrease simulation speed
- `C`: settings
- `ESC`: pause menu / quit
